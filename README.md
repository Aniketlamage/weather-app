# WeatherApp

Project created to learning basics of angular, Npm, node, HTML, CSS,Typescript, BOOTSRAP

## Versions

- **angular/cli:** 12.2.3
- **angular/core:** 12.2.3
- **angular/compiler:** 12.2.3
- **node:** v14.16.1
- **npm:** 6.14.12

## Running the app

```bash
git clone https://github.com/Aniketlamage/weather-app.git
cd weather-app
npm install
ng serve
```
